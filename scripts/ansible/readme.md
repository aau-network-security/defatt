# Ansible Configuration

This folder contains set of configuration files for setting up an environment for defat. 
Only requirement is to run the playbook with given command below :

```bash 
$ ansible-playbook -i hosts playbook.yml
``` 

The playbook is only applicable for Debian based Linux machines which have `apt` package manager.
